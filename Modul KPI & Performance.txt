# Modul KPI & Performance

## Tujuan

Mengukur performa karyawan secara objektif, terukur, dan terintegrasi langsung ke Assessment Engine serta Payroll Engine.

---

# Hak Akses

## Superadmin

* Full Access

## Direksi

* Melihat seluruh KPI perusahaan
* Approval KPI
* Melihat ranking performa

## Manager

* Membuat KPI bawahan
* Melakukan penilaian
* Monitoring progress tim

## HR

* Mengelola template KPI
* Monitoring seluruh divisi
* Kalibrasi penilaian

## Karyawan

* Melihat KPI pribadi
* Update progress
* Melihat histori penilaian

---

* Satuan
* Periode
* Divisi
* Jabatan

| KPI                   | Target | Bobot |
| --------------------- | ------ | ----- |
| Ticket Closed         | 500    | 30%   |
| SLA Compliance        | 95%    | 40%   |
| Customer Satisfaction | 90%    | 30%   |

# Performance Grade

| Nilai  | Grade |
| ------ | ----- |
| 95-100 | A+    |
| 90-94  | A     |
| 80-89  | B     |
| 70-79  | C     |
| <70    | D     |

---

# Integrasi Payroll

Grade A+
→ Bonus 20%

Grade A
→ Bonus 15%

Grade B
→ Bonus 10%

Grade C
→ Tidak ada bonus

Grade D
→ Coaching Program
